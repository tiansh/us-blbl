[bilibili Show Pages](http://tiansh.github.io/us-blbl/bilibili_show_pages/)<br />
在 bilibili 上，鼠标长按视频链接显示视频各分页的链接，方便直接打开想要的分页

bilibili视频页面上，缩放浏览器窗口到小窗口时自动将播放器页面全屏

[bilibili Player NO SSL](http://tiansh.github.io/us-blbl/bilibili_player_no_ssl)<br />
使用 HTTP 页面显示 bilibili 的播放器而不是使用 HTTPS 页面，适用于一些因故不能访问 HTTPS 的情况
